  was 0/0 matched); loader filters via `hasZeroKeyTokens`. Manifest
  regenerated; distribution still 10/10/8/8/7/7. Test suite **581** (was
  578; +3 selection / adapter regression tests).

## Task 2 — v2.5-07: AGENTS.md downstream evaluation

Whether the rules `ai-memory` generates actually steer real agents during
real coding sessions. CCEB measures *extraction*; this measures
*compliance downstream*.

The full operational runbook lives in
[`bench/agents-md-eval/README.md`](../bench/agents-md-eval/README.md).
The 10 micro-tasks are at `bench/agents-md-eval/tasks/T01.md` …
`T10.md`; the controlled fixture is at
`bench/agents-md-eval/controlled-repo/AGENTS.md`; the drift guard is
`bench/agents-md-eval/scripts/verify-agents-md.ts`.

Sketch:

```bash
# 0. Pre-flight (drift guard)
npm run build
npx tsx bench/agents-md-eval/scripts/verify-agents-md.ts
# Expected: "[verify-agents-md] OK — frozen fixture matches ..."

# 1. Set up scratch copy (Windows PowerShell)
New-Item -ItemType Directory -Path $env:TEMP\ai-memory-eval -Force | Out-Null
Copy-Item -Recurse bench\agents-md-eval\controlled-repo\* $env:TEMP\ai-memory-eval\internal-tools\

# 2. Run 10 tasks in Cursor (~30 min) — verbatim prompts, fresh chat per task,
#    save responses to bench/agents-md-eval/results/cursor/T0X-response.md
# 3. Run same 10 tasks in Claude Code (~30 min)
# 4. Score the 50 rows in bench/agents-md-eval/scoring-template.csv
#    (25 obs/editor × 2 editors); save as bench/agents-md-eval/results/scores.csv
# 5. Publish: fill docs/agents-md-eval-results.md, propagate headline,
#    flip ROADMAP v2.5-07 checkbox.
```

**Strict-rubric reminder** (locked in spike §4): score `1` only when the
literal pattern in the observation matches; "agent did X-ish" scores `0`.
Do not relax post-hoc; if the rubric feels wrong, that's a re-spike, not
a scoring decision.

**Headline:** `X / 50 rule observations honored across Cursor + Claude Code`.
Surfaces to update on publish: README, `competitive-landscape.md`,
`v2.5-03-submission-packet.md` ("key features" blurb), CHANGELOG, ROADMAP.
